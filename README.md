# FPVCustomEngine_M5 — Custom WebGL2 FPV game + Android APK (GitHub Actions)

Самописный движок (TypeScript + WebGL2) + сборка **Android APK** через GitHub Actions (Capacitor).
Есть: сцены, магазин, 5 карт, 12 дронов, Acro/Angle, touch-стики, геймпад (PS4 профиль), 3 языка.

## Быстрый старт
```bash
npm i
npm run dev
```

## Android APK через GitHub Actions
1) Создай репо `FPVCustomEngine_M5` и запушь этот проект.
2) GitHub → Actions → **Android Build** → скачай artifact `FPVCustomEngine-Android` (там `app-debug.apk`).
